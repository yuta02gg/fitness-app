# テストケース

## 概要

Fitness Appの主要機能について、手動で動作確認を行いました。

## テスト環境

- OS：Windows 11 / WSL2
- ブラウザ：Google Chrome
- Ruby：3.2.3
- Ruby on Rails：7.2.3
- DB：PostgreSQL

## テスト結果

### 基本表示・認証機能

| No | テスト内容 | 期待結果 | 結果 |
|---|---|---|---|
| 1 | トップページにアクセスする | トップページが表示される | OK |
| 2 | 有効なメールアドレスとパスワードでユーザー登録する | ユーザー登録できる | OK |
| 3 | 登録済みユーザーでログインする | マイページに遷移する | OK |
| 4 | ログアウトボタンを押す | ログアウトできる | OK |
| 5 | ゲストログインボタンを押す | ゲストユーザーとしてログインできる | OK |

### マイページ・カレンダー機能

| No | テスト内容 | 期待結果 | 結果 |
